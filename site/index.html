Two halves, one window

It's not a chat box bolted onto an editor.

The editor and the agent share one workspace — same files, same git state. Both are first-class.

01 / THE EDITOR

A real IDE surface

CodeMirror 6 with desktop-grade tooling — nothing stubbed.

  • Language intelligence — LSP completions, diagnostics, hover, code actions, inlay hints. Auto-install for TS, Python, Rust, Go, C++, Java.
  • DAP debugger — breakpoints, stepping, variables, call stack. launch.json mirrors VS Code.
  • Integrated terminal — xterm.js, split tabs, full PTY.
  • Deep git — per-hunk staging, commit graph, inline blame, conflict resolution.
  • Peek & go-to-definition, multi-cursor, project-wide find & replace.
02 / THE INTELLIGENCE

An agent that ships

Claude at the center, wired into your repo, your tools, and real guardrails.

  • Multi-provider kernel — Claude, Bedrock, OpenAI, Gemini, Ollama. Auto-routing, cost tracking.
  • MCP native — stdio, SSE & HTTP transports, auto-discovery of existing configs.
  • Permission modes — safe / fast / unrestricted / plan, with per-tool rules.
  • Golems — per-machine named agents with persistent, queryable history.
  • PRD executor — multi-story plans, parallel executors, auto-commit on completion.
