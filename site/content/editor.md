# The editor

CodeMirror 6, full desktop tooling — nothing stubbed.

- **Language intelligence** — LSP completions, diagnostics, hover, code actions, inlay hints; auto-installed for TS, Python, Rust, Go, C++, Java.
- **Debugger** — DAP breakpoints, stepping, variables, call stack. `launch.json` mirrors VS Code.
- **Terminal** — xterm.js, split tabs, a full PTY.
- **Git** — per-hunk staging, commit graph, inline blame, conflict resolution.
- **Navigation** — peek & go-to-definition, multi-cursor, project-wide find/replace.
- **Live markdown** — Typora-style WYSIWYG for `.md`. The file you're reading renders the same way inside E.
