# Media, inline

E renders more than text — in the buffer and in the chat.

- **Inline media** — image paths and SVG literals preview right in the code.
- **Sparklines** — numeric array literals draw a tiny chart beside them.
- **Color & number widgets** — swatches and drag-to-change scrubbers on literals.
- **Rich chat** — diffs, tables, and tool output render in line.
- **Dependency canvas** — files as a 2D / 3D graph you fly through.
